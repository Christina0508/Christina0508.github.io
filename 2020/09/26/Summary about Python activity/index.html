Summary about python activity

作为一个编程小白，我在大概三个月前才开始学习python。起初的初衷是因为python如此的火爆，已经成为现如今流行的存在。由于未来专业方向的缘故，python的学习能为我未来的学习省下许多的力气。首先，Python是一门解释性语言，这是他的特点，这让Python似乎与“高效”无缘。然而你知道吗，著名的Instagram服务器就是是用Python+Django搭建的，而且月活用户超过7亿。要知道Python+Django是一个以速度慢著称的web框架，一定是python的某个优点深深的吸引了他们。是的，这个优点就是，开发效率高。

Python is a programming language that lets you work quickly and integrate systems more effectively.
–摘自Python官网www.python.org

由于在此之前，可以说我几乎从未接触过编程，所以在进行时，可是说是比较艰难，而且由于在平时的学习中几乎用不到编程，所以我练习的机会非常的少，所以对于知识点的运用非常的缺乏。经过了几乎一个暑假的荒废，我已经忘了很多的知识，所以抱着来唤醒回忆的目的来参加这个活动。

这个活动的流程主要是使用python的语言来分析博主的情感状态，这是用数据分析的方法来获得准确的情感状态。首先拿到这个题目的时候是我非常是一头雾水的，毫无头绪，因为我并不知道这是让我们干什么。还好有老师给我们分析这个题目，这个题目主要就是把单词和符号分离，然后把所有的句子分析归类，来找到表达情感状态的单词，从而来表达人物的情感状态。

在一开始，我只能先看老师的演示程序来试图理解这个分析的流程。通过程序的报错来发现问题，渐渐的，我开始慢慢回忆一些学过的知识点，所以我翻看书来回忆学过却遗忘的知识点，根据书上的例子来运用到我们的题目中去，我发现可以有效的回忆和使用以前遗忘的知识点来编写程序了。但是由于重新拾起的缘故，所以我在进行得过程中出现了很多无法自己解决的报错问题，在学长的细心指导下，我才能一直畅通的进行下去。

通过这次的活动，让我重新温习了一些已经遗忘了的知识。python是一门需要练习的学问，想要能够完美的学习会python，光靠老师的讲解和书上例题的演示是远远不够的，只有不断的练习巩固和操练才能真正的掌握。由于平时的学业，能够使用到python 的时机也是非常的少的，这样的活动不仅让我能巩固我的编程知识，还让我来认识和了解到大学的python作业是什么样的，对于我未来的申请肯定是有很大帮助的。我希望我能够多参加这样的活动，让我的python知识不是只存在于书本而是真正的对我起到发展性的作用。
